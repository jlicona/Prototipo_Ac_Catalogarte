Catalogarte por Ac

Este prototipo ha sido desarrollado con el fin de demostrar la funcionalidad que Ac propone al Instituto Nacional de Bellas Artes (INBA) en el contexto de Retos Públicos, segunda etapa.

----------------------- TECNOLOGÍA

El servidor hace uso del framework de código abierto CodeIgniter para el desarrollo de aplicaciones web basadas en PHP. La base de datos es MySQL. Se usan técnicas de Javascript asincrono con Json, la librería Jquery y la librería de código abierto Snap.svg de Adobe para la matriz de exposiciones. Otras tecnologías empleadas en el resto de la página son Javascript regular y CSS.

----------------------- ACCESO A LA ADMINISTRACION DEL SITIO

Por ahora se definen dos roles de usuario.  Por un lado, el Adminsitrador del Sistema, con perfil técnico y dedicado a la adminsitración de los usuarios, así como soporte técnico, y por otro, los Editores, dedicados a crear exposiciones y artículos.

El acceso para estos roles está en la siguiente página:

http://catalogarte.net/index.php/admin/acceso

Las credenciales de acceso son las siguientes:

Usuario:	Administrador
Contraseña:	admin

Usuario:	Editor
Contraseña:	editor
